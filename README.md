# Hotel Booking Cancellations Dashboard 📊

## 📌 Project Overview
This project is an **Excel-based interactive dashboard** that analyzes hotel booking and cancellation trends.  
It provides insights into booking patterns across **City Hotels** and **Resort Hotels**, guest types, and seasonal variations.

The dashboard was created by learning from online resources (YouTube tutorials) and customized for practice and presentation.

---

## 🎯 Key Features
- **Total Bookings vs Cancellations** summary
- **Hotel Type Analysis**: City Hotel vs Resort Hotel
- **Guest Type Analysis**: Couples, Families, Singles
- **Desired vs Un-Desired Guests** comparison
- **Monthly Trends**: Bookings and cancellations across all months
- Interactive **year filter** (2015, 2016, 2017)

## 🚀 How to Use
1. Download the Excel file from this repository.
2. Open in **Microsoft Excel** (2016 or later recommended).
3. Use the slicers/filters to explore booking and cancellation data.
4. Analyze trends and patterns for insights.

---

## 📈 Insights from Dashboard
- Couples account for the majority of bookings and cancellations.
- City Hotels have higher bookings and cancellations compared to Resort Hotels.
- Seasonal peaks in bookings are visible, with cancellations varying month to month.
- Desired guests form the bulk of bookings, but cancellations are significant.
